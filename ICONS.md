# Icon Files Required

The extension requires icon files for the browser toolbar. You need to create three icon files:

- `icon16.png` - 16x16 pixels (toolbar icon)
- `icon48.png` - 48x48 pixels (extension management page)
- `icon128.png` - 128x128 pixels (Chrome Web Store)

You can create simple icons using any image editor, or use online tools like:
- https://www.favicon-generator.org/
- https://realfavicongenerator.net/

For a quick placeholder, you can create simple colored squares with the text "IN" (Inline) or use any icon design you prefer.

Alternatively, you can temporarily comment out the "icons" section in manifest.json to test the extension without icons.
